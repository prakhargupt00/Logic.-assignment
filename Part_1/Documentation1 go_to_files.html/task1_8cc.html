int main ( )

main function takes infix and outputs postfix,traverses parse tree to print back infix

Program starts here. An infix is taken from the user ,and a postfix is generated. Parse tree is made from postfix and inorder traversal is made to get back infix. If both the input infix and the infix from traversal of parsetree match then the formula is VALID and prints "MATCHING". Else if both do not match then the function prints "NOTMATCHING" which indicates an invalid formula.
